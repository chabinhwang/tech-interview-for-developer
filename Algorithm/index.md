# 🧮 Algorithm (알고리즘)

프로그래밍의 기초이자 코딩 테스트에서 가장 많이 등장하는 **정렬, 탐색, 그리고 알고리즘 설계 기법**을 다루는 공간입니다.
좌측 사이드바를 통해 아래의 세부 카테고리별 개념을 확인하실 수 있습니다.

### 📚 세부 카테고리
*   **기본 정렬:** 거품 정렬, 선택 정렬, 삽입 정렬 등 기본적인 정렬 원리
*   **고급 정렬:** 퀵 정렬, 병합 정렬, 힙 정렬, 기수 정렬 등 효율적인 정렬 알고리즘
*   **탐색 알고리즘:** 이분 탐색, BFS(너비 우선 탐색) & DFS(깊이 우선 탐색) 등 그래프 탐색 기법
*   **고급 설계 기법:** 동적 계획법(DP), 다익스트라(최단 경로), 최소 공통 조상(LCA), 비트마스크 등 복잡한 문제 해결 전략
*   **해시(Hash) 구현:** 해시 테이블의 충돌 해결 기법과 동작 원리
