# 🌐 Web (웹 프로그래밍)

브라우저의 동작 원리부터 REST API, OAuth, 세션/쿠키, 프론트엔드 프레임워크(React/Vue) 및 백엔드(Spring/JPA)까지 **웹 생태계의 전반적인 기술 지식**을 다루는 공간입니다.
좌측 사이드바를 통해 아래의 세부 카테고리별 개념을 확인하실 수 있습니다.

### 📚 세부 카테고리
*   **Web Basics (기초):** 브라우저 렌더링 방식, HTTP 프로토콜, 보안(XSS/CSRF), JWT, 쿠키 등 웹 기본 기술
*   **Spring (스프링):** Spring MVC, Boot, Security, JPA(Dirty Checking) 등 엔터프라이즈 백엔드 아키텍처
*   **React / Vue:** SPA, CSR vs SSR, 컴포넌트 라이프사이클, 훅(Hook) 등 최신 프론트엔드 프레임워크 원리
*   **DevOps:** Travis CI, 시스템 규모 확장, 로드밸런싱 등 인프라 및 배포 전략
